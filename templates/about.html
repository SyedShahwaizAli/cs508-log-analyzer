Secure Cloud Log Analyzer

About this project

This cloud log analyzer is designed to help students build a unique, hands-on CS-508 project with:

  • Multi-cloud log parsing: AWS CloudTrail, Azure Activity, GCP Audit, plus generic text logs.
  • Suspicious event detection and risk scoring.
  • Web-based upload UI and summary dashboard.
  • Extensible Python structure so you can add new rules or deploy to containers.

Unique features you can customize

  • Add a real IP reputation list for stronger alerts.
  • Integrate a cloud API to fetch logs from S3 or Azure Blob Storage.
  • Extend analysis with anomaly detection using frequency or login patterns.

Project author: CS-508 student. Teacher-facing note: includes custom parsing and cross-cloud security alert scoring.
